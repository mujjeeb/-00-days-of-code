# Higher or Lower Game

## Project Overview

The "Higher or Lower Game" is a Python-based text-based game that challenges players to guess which of two pre-listed celebrities has a higher follower count on a social media platform. The player continues playing until they make an incorrect guess. This project provides an entertaining way to test your knowledge of celebrities' popularity on social media.

## Project File

The project consists of the following file:

- `Day_13_Higher_or_lower_game.ipynb`: The Jupyter Notebook file containing the Python code for the game.

## Game Rules

Here are the rules and characteristics of the Higher or Lower Game:

- The game provides two pre-listed celebrities and their respective follower counts on a social media platform.
- The player's objective is to guess which of the two celebrities has a higher follower count.
- The player can continue playing as long as they make correct guesses.
- Once the player makes an incorrect guess, the game ends, and their final score is displayed.

## Prerequisites

To play the "Higher or Lower Game," you need to have Python and Jupyter Notebook installed on your system. You can download Python from the official [Python website](https://www.python.org/downloads), and Jupyter Notebook can be installed using `pip`.

## Getting Started

To start playing the "Higher or Lower Game":

1. Clone this project repository to your local machine:

   ```bash
   git clone https://github.com/yourusername/Higher_or_Lower_Game.git
   ```

2. Change your working directory to the project folder:

   ```bash
   cd Higher_or_Lower_Game
   ```

3. Open the Jupyter Notebook, "Day_13_Higher_or_lower_game.ipynb," using Jupyter Notebook:

   ```bash
   jupyter notebook Day_13_Higher_or_lower_game.ipynb
   ```

4. Follow the instructions provided in the notebook to start and play the game.
